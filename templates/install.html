Install

The following steps are needed in order to be able to play the mod.

First, download the Source SDK Base 2013 Multiplayer. You can click here if you have Steam opened and it will start the download.

Once it's installed, you want to go to its Betas window, so right click on it on your library, select properties, and then Betas. You will need beta_test in order to play the mod.

Once it's installed, download the mod files from here. Download the file "momentum.zip" and make sure to extract it inside Steam/Steamapps/sourcemods/

Finally, restart Steam and you should see Momentum Mod listed on your Steam library ready to be played!
