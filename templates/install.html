Install

Currently, you can play the prealpha builds of the mod as they are updated on GitHub.

First, download the Source SDK Base 2013 Multiplayer. You can click here if you have Steam opened and it will start the download.

Once it's installed, you want to go to its Betas window, so right click on it on your library, select properties, and then Betas. You will need beta_test in order to play the mod.

Once it's installed, you can get the latest release from here. Download the file "momentum-<version>.zip" and make sure to extract it inside Steam/Steamapps/sourcemods/

Finally, restart Steam and you should see Momentum Mod listed on your Steam library ready to be played! Remember that we are currently in Pre-Alpha state, and as such the current build is not reflective of final quality. We plan to add many more features before we release the mod!
